# Mixed Review Lens

Check both technical and business failure modes:

- code outputs feed the right business deliverables;
- generated artifacts can be reproduced;
- source data is not overwritten;
- validation includes commands and stakeholder acceptance;
- snapshots cover code, data, and documents;
- privacy constraints apply across data and generated artifacts;
- rollout or delivery steps are ordered safely.

Block execution if the plan proves code correctness but not business usefulness, or business acceptance but not code correctness.
